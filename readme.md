1. 파이썬 (Backend) 환경 설치
파이썬 라이브러리는 터미널에서 아래 명령어로 설치합니다.

Flask: 웹 서버 구축용

Flask-CORS: 프론트엔드(React)와 백엔드 간의 통신 허용용 (필수!)

Pillow: 이미지 생성용 (PIL의 최신 버전)

참고: uuid, random, io, base64는 파이썬 기본 내장이라 따로 설치 안 해도 됩니다.

Bash
pip install flask flask-cors Pillow
2. Node.js (Frontend) 환경 설치
프론트엔드 폴더(React 프로젝트 폴더)로 이동해서 설치해야 합니다.

Node.js 설치: 공식 홈페이지에서 설치되어 있어야 합니다.

axios: 백엔드와 통신하기 위한 라이브러리입니다.

Bash
# 프론트엔드 폴더로 이동 후
npm install axios
